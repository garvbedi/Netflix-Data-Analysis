# 🎬 Netflix Content Dashboard

Interactive dashboard analyzing 8,000+ Netflix titles to uncover content strategy trends.

## 🔗 Live Demo
https://netflix-data-analysis-awich6kzzqwnwxk32j479v.streamlit.app/

## Key Insights
- Netflix added most content between 2018–2020
- Movies dominate over TV Shows
- USA and India are top content producing countries

## Tech Stack
Python, Pandas, Plotly, Streamlit
